
# ✅ TaskMaster Pro

**TaskMaster Pro** is a beautiful, feature-rich task management system to help you organize your life, increase productivity, and never miss a deadline.

![TaskMaster Screenshot](https://github.com/ritesh5102/To-Do-List/blob/main/Screenshot%20(348).png)

## 🧰 Features

- 📋 Add, Edit, and Delete Tasks
- 🎯 Task Prioritization (High, Medium, Low)
- 📂 Task Categorization
- 🟢 Active, Completed, and Overdue Filters
- 📊 Task Status Summary (Total, Completed, Active, Overdue)
- 🧼 Clean and modern UI
- 📱 Responsive Design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla or React – depending on implementation)
- Optionally TailwindCSS / Bootstrap

## 🧪 How to Use

1. **Clone the Repository**
   ```bash
   git clone https://github.com/ritesh5102/To-Do-List.git
   cd To-Do-List-pro
   ```

2. **Run the App**
   Open `index.html` (or run `npm start` if using React) to launch in browser.
